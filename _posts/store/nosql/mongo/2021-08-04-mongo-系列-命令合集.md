* 查看当前指向的数据库 
``` > db ```

* 选择数据库 
``` > use foobar ```

* 插入数据
``` > db.blog.insert(post) ```

* 查找数据
``` > db.blog.find() ```

* 查找一个数据
``` > db.blog.findOne() ```

* 更新数据
``` > db.blog.update({"title":"My Blog Post"}, post) ```

* 删除数据
``` > db.blog.remove({"title":"My Blog Post"}) ```
