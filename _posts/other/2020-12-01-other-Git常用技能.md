1. 删除远程分支: git push origin -d 分支名  // git push origin --delete branch

2. 删除本地分支: git branch -d 分支名

3. 查看远程分支和本地分支的对应关系:git remote show origin  

4. 删除远程已经删除过的分支:git remote prune origin
